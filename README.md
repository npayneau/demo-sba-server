# Demo Spring Boot Admin

Démo de mise en place d'un Spring Boot Admin Server

## Pré-requis

Disposer d'un JDK8

## Compilation

Lancer la commande suivante:

```
mvn clean install
```

## Execution

Lancer la commande suivante:

```
java -jar core/target/core.jar
```

Utiliser l'URL suivante pour accèder à votre console d'administration
